# Trivia

Quizz game for discord built with nodeJs

## Getting Started

Getting all needed packages
```
npm install
```

Launch node server
```
node bot.js
```

## Built With

* [NodeJs](https://nodejs.org/en/) - Server side
* [DiscordJs - ~11.1.0](https://discord.js.org/#/) - Node module to interact with Discord API
